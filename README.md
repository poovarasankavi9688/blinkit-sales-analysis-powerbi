# 🛒 Blinkit Sales Analysis Dashboard | Power BI

## 📌 Project Overview
This project focuses on analyzing Blinkit's sales performance, outlet trends, item categories, and customer insights using an interactive Power BI dashboard. The dashboard provides business intelligence through KPIs, charts, and filters to help understand sales performance and outlet efficiency.

---

## 🎯 Business Problem
Blinkit operates across multiple outlets with different item categories and outlet types. The objective of this dashboard is to:

- Analyze overall sales performance
- Understand outlet performance by location and size
- Identify top-performing item categories
- Track outlet establishment trends
- Evaluate customer ratings and sales efficiency
